# TaskFlow 📱

**مهامك تحت السيطرة**

تطبيق ويب تقدمي (PWA) لإدارة المهام اليومية، مبني بـ React مع دعم العمل بدون إنترنت.

## ✨ المميزات

- 📝 إضافة وحذف المهام
- ✅ تحديد المهام المكتملة
- 💾 حفظ المهام تلقائياً في ذاكرة التطبيق
- 📱 تثبيت التطبيق على الهاتف
- 🌐 العمل بدون إنترنت
- 🌙 واجهة مظلمة أنيقة
- 🇸🇦 دعم اللغة العربية

## 🚀 البدء السريع

### المتطلبات
- Node.js 14+
- npm أو yarn

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/alhusa1shmi-lang/taskflow.git
cd taskflow

# تثبيت المتطلبات
npm install
```

### تشغيل التطبيق

```bash
npm start
```

سيفتح التطبيق على `http://localhost:3000`

### بناء النسخة الإنتاجية

```bash
npm run build
```

## 🔧 التقنيات المستخدمة

- **React 18** - مكتبة واجهات المستخدم
- **PWA** - تطبيق ويب تقدمي
- **Service Worker** - للعمل بدون إنترنت
- **localStorage** - لحفظ البيانات

## 📱 تثبيت التطبيق على الهاتف

### على Android
1. افتح التطبيق في متصفح Chrome
2. اضغط على قائمة المتصفح (⋮)
3. اختر "تثبيت التطبيق"

### على iOS
1. افتح التطبيق في Safari
2. اضغط على زر المشاركة (⬆)
3. اختر "إضافة إلى الشاشة الرئيسية"

## 🎨 التخصيص

### تغيير الألوان
عدّل `manifest.json` و `App.css`:
- `theme_color`: لون شريط الحالة
- `background_color`: اللون الخلفي
- `#6c63ff`: لون العنصر الرئيسي

### تغيير الأيقونات
أضف صور PNG بأحجام 192x192 و 512x512 في مجلد `public/`
